周一食谱

当日总热量 ≈ 1400 大卡

早餐 · 燕麦豆浆 + 水煮蛋 (350 大卡)

核心食材：燕麦片、无糖豆浆、鸡蛋

  • 燕麦片加热水或小火煮至软
  • 鸡蛋水煮 8-10 分钟
  • 搭配无糖豆浆食用

营养亮点：高蛋白 + 高纤维，饱腹感强

午餐 · 清蒸鸡胸 + 西兰花 + 糙米 (500 大卡)

核心食材：鸡胸(150g)、西兰花(200g)、糙米(100g熟)

  • 鸡胸加少许盐蒸 10-12 分钟
  • 西兰花焯水后少油快炒
  • 糙米提前浸泡后蒸熟

营养亮点：高蛋白、维生素 C 丰富，控油少盐

晚餐 · 番茄虾仁 + 炒时蔬 + 紫薯 (520 大卡)

核心食材：虾仁(150g)、番茄、混合蔬菜、紫薯(150g)

  • 虾仁快炒至变色后装盘
  • 番茄切丁快速翻炒
  • 紫薯蒸熟

营养亮点：优质蛋白 + 抗氧化番茄红素

加餐 · 低脂原味酸奶 (120 大卡)

核心食材：低脂酸奶(150g)

  • 冷藏后开盖即食
  • 可加少量奇亚籽（可选）
  • 控制总量，避免过甜添加

营养亮点：益生菌 + 钙质补充
