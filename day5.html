周五食谱

当日总热量 ≈ 1370 大卡

早餐 · 杂粮粥 + 鸡蛋 + 少量榨菜 (340 大卡)

核心食材：燕麦/藜麦、鸡蛋、榨菜

  • 杂粮煮粥
  • 鸡蛋水煮或荷包蛋
  • 榨菜少量点缀

营养亮点：复合碳水 + 蛋白质，温和易消化

午餐 · 香菇青菜 + 瘦牛肉 + 荞麦面 (520 大卡)

核心食材：香菇、青菜、瘦牛肉(120g)、荞麦面(100g熟)

  • 香菇青菜快炒
  • 牛肉切片煎炒至熟
  • 荞麦面煮熟过凉拌

营养亮点：低脂主食 + 高蛋白组合

晚餐 · 蒜蓉西兰花 + 虾 + 糙米 (500 大卡)

核心食材：西兰花(200g)、虾(150g)、糙米(80g熟)

  • 西兰花蒜蓉清炒
  • 虾仁快炒至变色
  • 配糙米

营养亮点：维生素 C + 海鲜优质蛋白

加餐 · 无糖绿茶 / 黑咖啡 (10 大卡)

核心食材：茶 / 咖啡

  • 少量饮用，避免晚间影响睡眠
  • 避免加糖
  • 搭配白水补充水分

营养亮点：低热量提神
